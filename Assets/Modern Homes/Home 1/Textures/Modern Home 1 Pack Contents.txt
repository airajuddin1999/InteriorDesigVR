Modern Home 1 Pack contains the following folder
- Exterior
- Home 1
- Prefabs
- Scene

The Exterior folder consists of the following folders
- Materials 	- Consists of all the material for the exterior used in the pack
- Mesh 		- Consists of the FBXs used for exterior in the pack
- Textures 		- Consists of all the texture maps used for the exterior base

The Home 1 folder consists of the following folders
- Materials 	- Consists of all the material for Home 1 used in the pack
- Mesh 		- Consists of all the FBXs for Home 1 used in the pack
- Textures 		- Consists of all the texture maps for the Home 1 used in the pack
- PSD Textures	- Consists of all the Original texture with respective maps

The Prefabs folder consists of the following
- Includes the Modern Home 1 Prefab

The Scenes folder consists of the following
- W358_ReflectionProbes folder - Consists of all the Reflection Probes/Cubemaps for Home 1 used in the pack
- Also includes a save file of Modern Home 1
